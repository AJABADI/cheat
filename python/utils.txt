# Lambda
* Scope is evaluated when lambda is called, not when lambda is defined!

d = {}
for i in range(10):
  d[i] = lambda: i
d[0]() -> 9

d = {}
for i in range(10):
  d[i] = lambda x=i: x  // default variables are are evaluated with lambda is defined
d[0]() -> 0
